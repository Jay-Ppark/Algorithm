# 버블 정렬(Bubble Sort)  
* 버블정렬(idea: 옆에 있는 값과 비교해서 더 작은 값을 앞으로 보내면 어떨까?)  
* 비효율적  
## 오름차순으로 정렬하는 프로그램 작성  
Q1. 1 10 5 8 7 6 4 3 2 9  
```cpp
#include<iostream>
using namespace std;
int main(void){
    int arr[10]={1,10,5,8,7,6,4,3,2,9};
    for(int i=0;i<10;i++){
        for(int j=0;j<9-i;j++){
            if(arr[j]>arr[j+1]){
                int tmp=arr[j+1];
                arr[j+1]=arr[j];
                arr[j]=tmp;
            }
        }
    }
    for(int i=0;i<10;i++){
        cout<<arr[i]<<" ";
    }
    return 0;
}
```
* 비교연산 횟수: 10+9+8+ ... +1  
* 총 수행시간(일반식): N*(N+1)/2  
* 시간복잡도: O(N*N) 비효율적  
* 실제 수행시 선택정렬보다 느림  
* 이유: 계속 비교를 하면서 위치를 바꾸는 연산도 같이 하기 때문  
* 일반적으로 정렬 알고리즘 중 가장 느림
